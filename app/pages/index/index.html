    项目特性

    无需配置

    能够适应各类项目基础框架，无需配置Axios,Vuex,Router等基础模块

    快速上手

    项目提供的mixins,derective,scss等都有简单的示例,只需在原有文件的基础上进行改造、扩展即可满足各类需求

    敏捷开发

    包含多种命令行工具,提高生产力
    常见问题

    1、命令行工具怎么用？

    以压缩dist文件夹为例，在项目根目录运行npm run pack即可在项目根目录生成形如wwvue-cli_090945.zip这样的压缩文件，其他命令行工具详见

    2、怎么修改命令行工具满足个性化需求？

    命令行工具目前包含，仓库快速提交、生成目录树、生成自定义组件文件框架、dist压缩，具体文件均在build文件夹下，注释齐全可轻易完成改造

    3、Vuex模块化具体使用方式？

    Vuex封装在store文件夹下，入口为main.js，模块化使用方式可参考baseInfo

    在main.js中引入,后modules中添加

      modules: {
    baseInfoStore: baseInfo_store,
    //
  },
